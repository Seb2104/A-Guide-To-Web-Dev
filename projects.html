Build Amazing Projects


Put your HTML and CSS skills to the test with real-world projects. Build stunning websites and applications while learning best practices.


Mini Projects(Beginner)

Project 1:To-do List
  1. Make a simple list of tasks in HTML.
  2. Use CSS to style checkboxes and labels.
  3. Add hover effects for better UX.
Project 2:Personal Portfolio Page
  1. Create an HTML file with a header, about section, and contact info.
  2. Style it with CSS (fonts, colors, layout).
  3. Add links to your social media or email.
Project 3: Recipe Page
  1. Create a page with a recipe title and image.
  2. Include ingredients in a bullet list.
  3. Write cooking steps in numbered order.

Practice Challenges(intermediate)

Challenge 1:Instructions Drop Down page
  1. Build a FAQ page where eac question expands and collapses when clicked
  2. Style it wth transitions and icons
Challenge 2:Product landing page
  1. Design a fake shop/website
  2. Sections:Hero header,Features,Pricing,Contact Form
  3. Use HTML5 form validation rules
Challenge 3:Responsive Weekly Planner
  1. Weekly timetable where you layout your activities for each day
  2. Style it like a calender grid
  3. Export/print friendly layout

Practice Challenges(Advanced)

Challenge 1:Responsive Grid layout
  1. Create a pintrest like layout
  2. Use Css grids or Flexbox
  3. Make it responsive so items rearrange dynamically on smaller screens
Challenge 2:Interactive Product Card
  1. Make a CSS only product card that flips when hovered
  2. Add animations using transform and transition
Challenge 3:Game Board
  1. Design a chessboard, tic-tac-toe grid, or Sudoku board using only HTML + CSS.
  2. Style it beautifully with hover states and custom colors.
