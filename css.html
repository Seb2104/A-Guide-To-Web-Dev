Introduction to CSS

what is CSS?


CSS stands for Cascading Style Sheets. It is the language used to describe the presentation of web pages, such as colors, layout, and fonts, in order to make our web sites more user friendly.


CSS is intended to create style sheets for the web. It is HTML-independent and compatible with any XML-based markup language. Now, let's attempt breaking the acronym:

  • Cascading: Falling of Styles
  • Style: Aadding designs/styling our HTML tags
  • Sheets: Writing our style in different documenets

How to apply CSS to your HTML file

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Priority Order

Inline > Internal > External

Inline CSS

  • Before CSS this was the only way to apply styles
  • Not an efficient way to write as it has a lot of redundancy
  • Self-contained
  • Uniquely applied on each element
  • The idea of separation of conerns was lost

Internal CSS

  • With the help of style tag, we can apply styles within the HTML file
  • Redundancy is removed
  • But the idea of separation of corncerns still lost
  • Applied on the same document as the HTML file

External CSS

  • Using <link ≶tag in the head of the page we can aplly these styles
  • Reference is added
  • File saved with .css extension
  • No redundacy present
  • The idea of separation of corncerns iis maintained
  • Uniquely applied to each document

    CSS Colors

    There are different colouring schemes in CSS, the three widely known techiques are:

    RGB

    • This starts with the letters RGB and takes 3 parameter values right after
    • The 3 parameters correspond to red, green and blue respectively
    • The value for each parameter may vary from 0 to 255

    HEX

    • HEX code starts with a # and is follwed by 6 numbers which are divided into sets of 3
    • Each set corresponds to the respective colours Red, Green and Blue
    • Single set value can vary from 00 to 09 and AA to FF

    RGBA

    • This starts with RGB then it takes 4 parameters
    • The fourth parameter corresponds to alpha
    • Value of the first three parameters may vary from 0 to 255 and the last parameter ranges from 0 to 1 that is from 0.1, 0.2,â€¦..0.9
